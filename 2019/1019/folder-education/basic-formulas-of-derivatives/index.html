基本函数导数及四则运算

  1. $c’ = 0$

  2. $(x^n)’ = nx^{n-1}$

    • $(\sqrt x)’ = \dfrac{1}{2\sqrt x}$

    • $(\dfrac{1}{x})’ = -\dfrac{1}{x^2}$

  3. $(\sin x)’ = \cos x$

  4. $(\cos x)’ = -\sin x$

  5. $(\ln x)’ = \dfrac{1}{x}$

  6. $(\log_a x)’ = \dfrac{\log_a e}{x}$

  7. $(a^x)’ = a^x\ln a$

    • $(e^x)’ = e^x$

  8. $(\tan x)’ = \sec ^2 x$

  9. $(\cot x)’ = -\csc^2 x$

  10. $(\sec x)’ = \sec x\tan x$

  11. $(\csc x)’ = -\csc x\cot x$

  12. $(\arcsin x)’ = \dfrac{1}{\sqrt{1 - x^2}}$

  13. $(\arccos x)’ = -\dfrac{1}{\sqrt{1 - x^2}}$

  14. $(\arctan x)’ = \dfrac{1}{1 + x^2}$

  15. $(arccot x)’ = -\dfrac{1}{1 + x^2}$

$(x^n)’ = nx^{n-1}$

由导数的定义：$f’(x)=\lim_{\Delta x\to 0}\dfrac{f(x + \Delta x) - f(x)}{\Delta x}$

$(\sin x)’ = \cos x$

$(\cos x)’ = -\sin x$

$(\ln x)’ = \dfrac{1}{x}$

$(\log_a x)’ = \dfrac{\log_a e}{x}$

$(a^x)’ = a^x\ln a$

$(\tan x)’ = \sec ^2 x$

$(\cot x)’ = -\csc^2 x$

$(\sec x)’ = \sec x\tan x$

$(\csc x)’ = -\csc x\cot x$

$(\arcsin x)’ = \dfrac{1}{\sqrt{1 - x^2}}$

设 $y = \arcsin x$，则$x = \sin y$

$(\arccos x)’ = -\dfrac{1}{\sqrt{1 - x^2}}$

设 $y = \arccos x$，则$x = \cos y$

$(\arctan x)’ = \dfrac{1}{1 + x^2}$

设 $y = \arctan x$，则$x = \tan y$

$(\rm{arccot} x)’ = -\dfrac{1}{1 + x^2}$

设 $y = arccot x$，则$x = \cot y$
